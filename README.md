# Polynomial Interpolation Calculator

This is a simple interpolantation graphing calculator based on a user inputted list of points. It has a very simple GUI, and is very easy to use

Written in Python

## Usage

Simply run main to start. Then, follow instructions on screen

## Backstory

I decided to create this project for three reasons:  
1. To practice my python skills
2. To work on my code-based implementation skills of math concepts
3. To get some extra practice with polynomial interpolation
